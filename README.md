# Site EGAD Contabilidade

Site institucional pronto para publicação na Vercel.

## Como publicar na Vercel

1. Crie uma conta no GitHub: https://github.com
2. Crie um repositório chamado `egad-contabilidade`
3. Envie todos os arquivos desta pasta para o repositório
4. Crie uma conta na Vercel: https://vercel.com
5. Clique em **Add New Project**
6. Escolha o repositório `egad-contabilidade`
7. Clique em **Deploy**

## Rodar no computador

```bash
npm install
npm run dev
```

Depois abra: http://localhost:3000
